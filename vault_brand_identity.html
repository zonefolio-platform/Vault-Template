01 · Concept

The idea

Vault has no fixed color identity. You bring the accent — the template adapts everything around it.

Name
Vault
"Where serious work lives."
Core premise
Your work is the identity. Vault uses a near-black editorial base with a user-defined accent color that runs through CTAs, highlights, borders, hover states, and the SplashCursor fluid. Change the accent and the entire template transforms — from cold electric blue to warm amber to hot coral. The structure stays the same. The soul is yours.
Who it's for
Developers, designers, photographers, motion artists, architects, copywriters — anyone whose work needs to be experienced, not just listed. The dark base makes every piece of work pop. The accent makes it personal.
Visual personality
Dark editorial. Magazine-grade. Confident without being loud. The near-black background makes work images the hero of every section. JetBrains Mono for technical metadata signals craft. The aurora and SplashCursor add life without noise.

06 · Section spec

Template sections

Five sections. Each with a distinct structural grammar — no two sections feel the same.

01
Hero
Full-bleed Aurora background. Centered or left-aligned display name with outline second line. RotatingText role subtitle. Scroll indicator. No profile image card — name IS the visual.

Work
Asymmetric grid — 1 large featured card (60% width) + 2 stacked smaller cards (40%). Project image fills top 60% of card. Hover reveals accent border glow. Stack in JetBrains Mono.
Asymmetric grid Accent glow hover JetBrains Mono tags
03
About
Two-column: long-form editorial bio left, curated stats right (years exp, projects shipped, clients). No "See more" toggle — the full bio is always visible. Silk bg option.

04
Stack
Horizontal scrolling tag cloud on Particles background. No boxes, no cards, no progress bars — floating skill names only. Accent color text, JetBrains Mono. Moves on hover.

Single centered column. Email address as a massive display-size link — nothing else. Social icons as small mono text below. No cards, no icon grids, no phone number. Just the email and an invitation.

07 · CSS variables

Full token reference

All values as CSS variables. The accent is the only one set at runtime from user preference. Everything else is static.

--vault-surface-raised:   #161B27;    /* elevated cards, hover */
--vault-border:           rgba(255,255,255,0.07);
--vault-border-mid:       rgba(255,255,255,0.13);

08 · Voice & copy

Copy tone

Vault should feel like a senior creative who doesn't need to sell themselves. Confident. Precise. Nothing wasted.

Write like this
Short section labels in mono — "Selected Work", "About", "Stack"
Contact email as a display-size link — no label needed
Bio: two sentences, present tense, specific — no fluff
Project names that describe what the project does
Stack in JetBrains Mono — plain list, no icons
Sentence case everywhere — always
Never write this
"Welcome to my portfolio" — it's dead weight
Skill progress bars with invented percentages
"Passionate about creating seamless experiences"
All-caps section headers — sentence case only
Contact forms instead of a direct email link
More than 3 lines of bio in the hero area
